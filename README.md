# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data from a telecom company and builds a machine learning model to predict whether a customer will leave the service.

## Objective
- Identify factors driving customer churn.
- Build a predictive model.
- Recommend retention strategies.

## Dataset
IBM Telco Customer Churn Dataset.

## Tools Used
- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Building
5. Evaluation
6. Business Recommendations

## Key Findings
- Month-to-month customers had the highest churn rate.
- Customers with short tenure churned more.
- Higher monthly charges were associated with higher churn.
- Tech support and online security reduced churn.

## Model Performance
| Metric | Value |
|------:|:------|
| Accuracy | 0.80 |
| Precision | 0.65 |
| Recall | 0.58 |
| F1 Score | 0.61 |
| ROC-AUC | 0.836 |

## Business Recommendations
- Encourage annual contracts.
- Offer retention discounts to new customers.
- Bundle tech support and security services.
